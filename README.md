## Info
This project contains 5 math based games.

## Requirements

Node.js : `^22.13.0`

## Installation

1. Clone the repository.
```shell
git clone https://github.com/Droletor/frontend-project-44.git
```

2. Install the dependencies.
```shell
make install
```

## Running the games

Using ``npm link``:

```shell
npm link
```

```shell
brain-games
```

Using make:

```shell
make brain-games
```
